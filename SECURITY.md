## Reporting a Vulnerability

We take security vulnerabilities seriously.

If you discover a security vulnerability within Express Tool, please send an e-mail to [INSERT EMAIL ADDRESS]. All security vulnerabilities will be promptly addressed.

**Please do not report security vulnerabilities through public GitHub issues.**

### Process

1. **Report**: You privately report the vulnerability to the maintainers.
2. **Triaging**: We will acknowledge receipt of your report and verify the vulnerability.
3. **Fixing**: We will work on a fix in a private environment.
4. **Disclosure**: Once the fix is ready and released, we will publicly disclose the vulnerability and credit you for the discovery (if you wish).

Thank you for improving the security of Express Tool!
